Pengantar

JavaScript adalah bahasa scripting lintas-platform, berorientasi objek. Ini adalah bahasa yang kecil dan ringan. Di dalam lingkungan host (misalnya, browser web), JavaScript dapat dihubungkan ke objek di lingkungannya untuk menyediakan kontrol terprogram terhadapnya.

JavaScript berisi pustaka objek standar, seperti Array, Tanggal, dan Matematika, dan kumpulan inti elemen bahasa seperti operator, struktur kontrol, dan pernyataan. JavaScript inti dapat diperluas untuk berbagai tujuan dengan melengkapinya dengan objek tambahan; Misalnya:

  • JavaScript sisi klien memperluas bahasa inti dengan menyediakan objek untuk mengontrol browser dan Document Object Model (DOM). Misalnya, ekstensi sisi klien memungkinkan aplikasi untuk menempatkan elemen pada formulir HTML dan merespons peristiwa pengguna seperti klik mouse, masukan formulir, dan navigasi halaman.
  • JavaScript sisi server memperluas bahasa inti dengan menyediakan objek yang relevan untuk menjalankan JavaScript di server. Misalnya, ekstensi sisi server memungkinkan aplikasi untuk berkomunikasi dengan database, memberikan kontinuitas informasi dari satu pemanggilan ke aplikasi lainnya, atau melakukan manipulasi file di server.

Apa Yang Harus Anda Ketahui

Panduan ini menganggap Anda memiliki latar belakang dasar berikut:

  • Pemahaman umum tentang Internet dan World Wide Web (WWW).
  • Pengetahuan kerja yang baik tentang HyperText Markup Language (HTML).
  • Beberapa pengalaman pemrograman. Jika Anda baru dalam pemrograman, cobalah salah satu tutorial yang ditautkan di halaman utama tentang JavaScript.

JavaScript dan Java

JavaScript dan Java serupa dalam beberapa hal tetapi secara fundamental berbeda dalam beberapa hal lainnya. Bahasa JavaScript menyerupai Java tetapi tidak memiliki pengetikan statis Java dan pengecekan tipe yang kuat. JavaScript mengikuti sebagian besar sintaks ekspresi Java, konvensi penamaan, dan konstruksi alur kontrol dasar yang merupakan alasan mengapa namanya diganti dari LiveScript ke JavaScript.

Berbeda dengan sistem kelas waktu kompilasi Java yang dibuat dengan deklarasi, JavaScript mendukung sistem waktu proses berdasarkan sejumlah kecil tipe data yang mewakili nilai numerik, Boolean, dan string. JavaScript memiliki model objek berbasis prototipe daripada model objek berbasis kelas yang lebih umum. Model berbasis prototipe menyediakan pewarisan dinamis; yaitu, apa yang diwariskan dapat bervariasi untuk masing-masing objek. JavaScript juga mendukung fungsi tanpa persyaratan deklaratif khusus. Fungsi dapat menjadi properti objek, mengeksekusi sebagai metode yang diketik secara longgar

JavaScript adalah bahasa bentuk yang sangat bebas dibandingkan dengan Java. Anda tidak harus mendeklarasikan semua variabel, kelas, dan metode. Anda tidak perlu memikirkan apakah metode bersifat publik, pribadi, atau dilindungi, dan Anda tidak perlu mengimplementasikan antarmuka. Variabel, parameter, dan tipe pengembalian fungsi tidak diketik secara eksplisit.

Halo Dunia

Untuk mulai menulis JavaScript, buka Scratchpad dan tulis kode JavaScript "Halo dunia" pertama Anda:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Pilih kode di pad dan tekan Ctrl+R untuk melihatnya terbuka di browser Anda!

Variabel

Anda menggunakan variabel sebagai nama simbolis untuk nilai dalam aplikasi Anda. Nama-nama variabel, yang disebut pengidentifikasi, sesuai dengan aturan tertentu

Pengidentifikasi JavaScript harus dimulai dengan huruf, garis bawah (_), atau tanda dolar ($); karakter selanjutnya juga bisa berupa angka (0-9). Karena JavaScript peka terhadap huruf besar/kecil, huruf menyertakan karakter "A" hingga "Z" (huruf besar) dan karakter "a" hingga "z" (huruf kecil).

Anda dapat menggunakan huruf ISO 8859-1 atau Unicode seperti å dan ü dalam pengidentifikasi. Anda juga dapat menggunakan urutan pelolosan Unicode sebagai karakter dalam pengidentifikasi. Beberapa contoh nama resmi adalah Number_hits, temp99, dan _name.

Mendeklarasikan variabel

Anda dapat mendeklarasikan variabel dengan tiga cara:

Dengan kata kunci var. Misalnya,

var x = 42.

sintaks ini dapat digunakan untuk mendeklarasikan variabel lokal dan global.

Dengan hanya memberikan nilai. Misalnya,

x = 42

Ini selalu mendeklarasikan variabel global. Ini menghasilkan peringatan JavaScript yang ketat. Anda sebaiknya tidak menggunakan varian ini.

Dengan kata kunci biarkan. Misalnya,

let y = 23
